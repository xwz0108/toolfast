### #1 Lucide — Best for Modern Apps

Clean, consistent icon set forked from Feather Icons. Lightweight SVG. First-class React and Vue component libraries. MIT licensed.

### #2 Phosphor — Best Variety

With 7000+ icons in 6 styles (thin, light, regular, bold, fill, duotone), Phosphor covers almost every use case. Great for design-heavy apps.

### #3 Material Icons (MUI)

If you are using MUI (Material UI), these are included by default. 3000+ icons covering most common UI patterns.

### Pro Tip

Stick to one icon library per project. Mixing icons from different sets creates visual inconsistency.
